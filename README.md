# AI Puzzle Designer

An AI-powered AI Puzzle Designer tool built with Next.js 16, TypeScript, Tailwind CSS, and DeepSeek API.

## Features
- Generate AI Puzzle Designer content instantly
- Dark gradient UI
- Powered by DeepSeek AI
- Fully responsive

## Setup
```bash
export OPENAI_API_KEY=your_deepseek_api_key
npm install && npm run dev
```
